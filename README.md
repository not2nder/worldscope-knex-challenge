# WorldScope

WorldScope é uma aplicação web desenvolvida para o desafio de Desenvolvedor Front-end do Processo da KNEX Consultoria Jr.

A aplicação tem como objetivo permitir a consulta, filtragem e visualização de informações sobre países, consumindo dados da API RestCountries.

## Tecnologias utilizadas

- Vite
- React
- TypeScript
- Tailwind CSS
